# 🥋 BJJ Protocol (V2)
### Sistema Full Stack de Gestão Pedagógica e Comunitária para Academias de Jiu-Jitsu

O **BJJ Protocol** é uma solução completa desenvolvida para transformar a gestão de academias, indo além de um simples CRM financeiro. O foco do projeto é a retenção de alunos através de dados, gamificação e comunicação automatizada.

> **Status do Projeto:** Em desenvolvimento (V2) 🚀  
> Este repositório serve como documentação técnica e vitrine de funcionalidades, visto que o código-fonte principal é privado.

## 🛠️ Tecnologias Utilizadas

O ecossistema do projeto foi construído utilizando as ferramentas mais modernas do mercado para garantir escalabilidade e performance:

* **Frontend:** React.js, Vite, TypeScript, Tailwind CSS.
* **Backend:** Node.js, Express.
* **Banco de Dados:** PostgreSQL (Hospedado via Neon DB).
* **ORM:** Drizzle ORM.
* **Infra:** Vercel (Frontend) & Render (Backend).

---

## 🌟 Funcionalidades em Destaque

### 1. Motor de Regras para Graduação
Um sistema inteligente que analisa métricas de presença, desempenho e tempo de casa para sugerir automaticamente a troca de faixas ou graus, eliminando a falha humana no acompanhamento do aluno.

### 2. Automação de Mensagens Pós-Aula
Integração que dispara feedbacks automáticos para alunos presentes e mensagens de "sentimos sua falta" para alunos ausentes, aumentando a taxa de retenção.

### 3. Comunidade Interativa
Um feed social interno onde alunos e professores podem interagir, postar conquistas e comentar sobre os treinos, fortalecendo o senso de pertencimento.

### 4. Gestão Comercial e Onboarding
Fluxo completo de entrada de novos alunos, desde o primeiro contato até a assinatura do contrato e o currículo pedagógico inicial.
